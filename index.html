Sound Ethics Capstone

Context

Question: Given a song, is it AI-generated or authentic? That was the question my capstone group tried to answer by building a detection model for our year-long project.

Approach

After reading research papers, we found generalization was a key challenge and expanded our dataset by generating additional music from multiple generators, then fine-tuned a classifier from scratch.

Results

Our generalized model improved substantially over the original baseline and performed well on previously unseen generators.

Next Steps

  • Continue expanding the dataset to improve generalization.
  • Explore finer-grained detection (e.g., identifying AI-generated segments).
  • Deploy the classifier for broader use by artists and platforms.

Movie Recommender System

Context

People have different tastes, and you don’t know whether a movie is “good” until you watch it. This project focused on recommending similar movies based on what a user already likes.

Approach

Using data from the TMDb API, we built a hybrid system: content-based recommendations for cold-start movies and collaborative filtering for movies with reviews.

Results

The system produced reasonable recommendations in qualitative tests, and the hybrid approach helped address the cold-start problem.

Next Steps

  • Incorporate additional content signals (e.g., production companies, keywords).
  • Expand review coverage by integrating additional sources.
  • Refine feature weighting and evaluate with user feedback.

Symbolic Music Generation

Context

We used the NES-MDB dataset (Nintendo game soundtracks) to generate symbolic music with an LSTM. The model predicts note sequences and outputs MIDI.

Approach

We implemented unconditioned generation and extended to conditioned harmony generation by feeding melody input and generating arpeggiated triads for a retro soundtrack feel.

Results

The LSTM outperformed a Markov chain baseline for unconditioned generation, and the harmonization preserved timing while following basic music theory.

Next Steps

  • Extend the symbolic representation to include rhythm and expressive dynamics.
  • Compare LSTM models with transformer-based architectures.
  • Develop evaluation methods that better align with human listening judgments.

Voter Data Analysis

Context

Voting is central to democracy, but turnout varies across regions and demographics. We worked with large-scale voter files to explore turnout patterns and potential contributing factors.

Approach

We cleaned, transformed, and analyzed large datasets and created visualizations (including maps) to interpret patterns in turnout and affiliation.

Results

We found patterns in turnout across geography and demographics and identified factors associated with participation differences.

Next Steps

  • Explore additional demographic and socioeconomic variables.
  • Expand analysis across multiple states/regions for comparison.
  • Investigate time trends and policy-relevant questions.

Music Genre Classification

Context

This project explored whether audio features could be used to classify a song’s genre using supervised ML models.

Approach

We compared several models (e.g., Random Forest, Gradient Boosted Trees) and tuned hyperparameters to improve performance.

Results

The best model achieved strong test accuracy and we analyzed misclassifications to understand where the model struggled.

Next Steps

  • Experiment with neural network models for richer representations.
  • Improve feature extraction and explore time-frequency features.
  • Use error analysis to guide targeted improvements.
